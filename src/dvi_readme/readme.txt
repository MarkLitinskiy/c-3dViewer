1.1 Описание
────────────

  3dViewer-v1.0 - это программа для просмотра 3D моделей, выполненная в
  рамках обучения в Школе 21. Данная программа позволяет открывать файлы
  формата .obj и просматривать их.


1.2 Зависимости
───────────────

  qt6+, qmake, openGL, gcc, make


1.3 Установка
─────────────

  Выполнение команды 'make install' в директории src установит программу
  в папку build


1.4 Использование
─────────────────

  Исполняемый файл находится в папке build. В центральной области
  программы находится 3d модель.

  Вкладка info содержит информацию о модели (кол-во вершин, рёбер,
  размер файла).

  Вкладка interation содержит поля для изменения отображения 3d модели
  (масштаб, перемещение по осям X, Y, Z, вращение по осям X, Y,
  Z). Вращение модели также можно осуществлять с помощью мыши (при
  зажатой ЛКМ), масштаб можно менять с помощью колёсика.

  Вкладка settings содержит настройки отрисовки модели (цвет заднего
  фона, вершин, рёбер, тип проекции (параллельная и центральная), тип
  отрисовки рёбер (сплошная линия, пунктирная линия), тип отображаемого
  чего-то ещё (круг, квадрат, ничего??). Настройки сохраняются при
  повторном открытии программы.

  Вкладка record содержит клавиши для сохранения скриншотов текущего
  отображения модели и для сохранения скринкаста (перемещение модели по
  заданным на вкладке "interation" параметрам, сохранённым в .gif
  документе.)
